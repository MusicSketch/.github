# Music Sketch 🎵

## 📌 프로젝트 소개

- 사용자의 음성 특성에 맞는 음악 추천 시스템
  1. librosa 라이브러리를 이용하여 사용자의 음성에서 melspectrogram 추출
  2. 해당 스펙트로그램을 이미지로 변환 후 학습된 모델에 적용하여 장르를 추천 받음
  3. 해당 장르의 음악 리스트 출력 및 미리듣기

## 📅 프로젝트 수행 기간

- 2022\. 09. 12 ~ 2022. 09. 30
